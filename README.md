# User Registration Form

This is a simple user registration form built using HTML.

## 🌐 Live Demo

https://abhishek-akkal.github.io/Html-form/

## 📌 Features

- Name, Email, Password & Confirm Password
- Gender selection
- Date of Birth
- City dropdown
- Address input

## ✅ Validation

- Required fields
- Proper email format
- Basic input validation

## 📧 Form Submission

- Uses `mailto` method
- Opens user's email client with filled data

## 🎯 Purpose

Built for learning HTML forms and basic validation.

## ⚠️ Note

This project uses only HTML (no backend), so email sending depends on the user's email client.
