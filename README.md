# User Registration Form

This is a simple user registration form built using HTML.

## 📌 Features

* Collects user details like:

  * Name
  * Email
  * Password & Confirm Password
  * Gender selection
  * Date of Birth
  * City (dropdown)
  * Address (textarea)

## ✅ Validation

* Required fields to prevent empty submissions
* Email input type for correct format
* Radio buttons for gender selection
* Date picker for easy DOB selection

## 📧 Form Submission

* Uses the `mailto` method
* Opens the user's default email client
* Automatically fills the entered data in the email body

## 🎯 Purpose

This project is created for learning purposes to understand:

* HTML forms
* Input types
* Basic form validation

## ⚠️ Note

This form uses basic HTML only and does not include backend functionality, so email sending depends on the user's email client.

## 🚀 Future Improvements

* Add CSS, JavaScript validation (e.g., password match)
* Improve UI with CSS
* Integrate backend for real form submission
